Tugas 2 - Mobile Development (GDGOC Telkom)
Repository ini berisi tugas "Hitung Total Belanja" menggunakan bahasa pemrograman Dart, yang mengimplementasikan konsep Object-Oriented Programming (OOP) dan Collections.

Fitur Code
✅ Penggunaan Class dan Object ( Properti nama dan harga )

✅ Implementasi Collection ( List untuk menampung objek )

✅ Perulangan ( for-in atau for loop untuk kalkulasi )

✅ Menampilkan Output ( Fungsi print() untuk total harga )

Detail Tugas
Tugas ini berfokus pada pengolahan data objek dalam sebuah daftar (list) untuk mensimulasikan sistem kasir sederhana:

Definisi Barang: Membuat class sebagai blueprint.

Manajemen Data: Memasukkan beberapa instance objek ke dalam List.

Algoritma: Mengiterasi list untuk menjumlahkan nilai properti harga dari setiap elemen.
